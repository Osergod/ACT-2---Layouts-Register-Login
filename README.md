# ACT-2---Layouts-Register-Login

Per a cada subactivitat, genera una nova Activity dins una mateixa App, accessibles des d'un menú principal amb buttons d'accès als diferents exercicis:

Exercici 6: Crea una pantalla amb un botó, i un EditText de manera que al apretar sobre el botó es mostri en un toast el valor recollit en el EditText.

Exercici 7: Crea una pantalla amb tres buttons, cadascun t'ha de dirigir a una activitat on de registre on l’usuari indicarà el seu nom, cognom,edat, usuari i password a¡mb un botó en la part inferior amb el text enviar, que simplement mostri un toast dient registre complet per pantalla. Per contenir els elements fes servir un LinearLayout. Realitza les altres versions d’aquesta pantalla fent servir un Relative Layout i un Constraint Layout.

Exercici 8: Crea un pantalla de login amb dos editText uno de tipo password i un altre de tipo plain text per al usuari, amb un botó de confirmació a la part de baix, que, en prèmer, faci un Toast amb i un Logcat amb la informació de usuari i contrassenya introduïts.

Exercici 9: Crea una pantalla de matriculació on podem triar entre diversas materies y torns per triar les materies fes servir un checkbox per triar torn de matí o tarda fes servir un radiobutton que només deixa triar una de les dues opcions. Afegeix un títol matriculació a la pantalla

Exercici 10: Crea una pantalla amb dos buttons: cadascuna porta a una diferent activitat: Una activitat amb tres botons situats horitzontalment fent servir un linearLayout, fes que l'últim botó ocupi el doble que els altres dos. La segona activitat que sigui tingui una interfície igual que l’anterior, però posicionant els botons en vertical i de manera que ara el primer sigui el que ocupa el doble que els altres dos. Estableix butons de navegació entre les activitat.
